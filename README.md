# wg-ai-alignment
This repo is for the AI Alignment for Open Source Working Group

What is AI Alignment?
AI alignment is the effort to design AI systems so their goals, behaviors, and decisions are consistent with human values. In open source, we’re focused on ensuring AI systems understand and respect what open source communities have built, what we know, what we value, and what we expect from AI systems who depend on open projects, and which turn up in those communities.

Please see the [CONTRIBUTING.md](CONTRIBUTING.md) file for more information on how to participate and contribute.
